📚 This repository contains **Web Technology Lab Programs** demonstrating the fundamentals of  
**HTML, CSS, JavaScript, Bootstrap, and PHP**.  
Each program helps beginners understand how to **design, style, and create interactive websites**.

---

# 📚 List of Programs

---

## 1️⃣ 🧾 Personal Details using HTML Text Formatting
This program creates a **simple HTML webpage** that displays personal details like **name, age, address, and hobbies**.

✨ Concepts Used  
- `<b>` Bold text  
- `<i>` Italic text  
- `<u>` Underline text  
- `<strong>` Important text  
- `<em>` Emphasized text  

🎯 Purpose: Learn **HTML text formatting tags**.

---

## 2️⃣ 🖼️ Webpage with Image, Hyperlink & Nested Table
This program demonstrates how to create a webpage that includes:

🖼️ **Image** using `<img>`  
🔗 **Hyperlink** using `<a>`  
📊 **Nested Table** (table inside another table)

🎯 Purpose: Understand **multimedia elements and table structures in HTML**.

---

## 3️⃣ 📝 HTML Form Creation
This program creates a **user input form** to collect information.

📥 Form Fields  
- 👤 Name  
- 📧 Email  
- 🚻 Gender  
- 🎂 Date of Birth  
- 🔘 Submit Button  

🎯 Purpose: Learn **HTML form elements and user input handling**.

---

## 4️⃣ 🎨 CSS Styling (Inline, Internal & External)
This program demonstrates **three methods of applying CSS**:

🟢 **Inline CSS** – Style inside HTML tags  
🟡 **Internal CSS** – Style inside `<style>` tag  
🔵 **External CSS** – Style in a separate `.css` file  

🎯 Purpose: Understand **different CSS styling techniques**.

---

## 5️⃣ 📱 Responsive Webpage using Bootstrap
This program creates a **mobile-friendly responsive webpage** using **Bootstrap**.

🚀 Features  
- Bootstrap **Grid System**  
- Responsive **layout design**  
- Components like **buttons, cards, and navbar**

🎯 Purpose: Learn **responsive web design**.

---

## 6️⃣ ⚡ JavaScript Form Validation
This program validates form inputs using **JavaScript**.

✔ Checks **empty fields**  
✔ Validates **email format**  
✔ Prevents invalid form submission  

🎯 Purpose: Improve **data accuracy and user experience**.

---

## 7️⃣ 🔄 Dynamic Content using JavaScript DOM
This program uses JavaScript to **manipulate webpage content dynamically**.

💡 Examples  
- Change text on button click  
- Add or remove HTML elements  
- Update webpage content dynamically  

🎯 Purpose: Learn **DOM manipulation**.

---

## 8️⃣ 🧮 Simple Calculator using JavaScript
This program creates a **basic calculator** using JavaScript.

➕ Addition  
➖ Subtraction  
✖ Multiplication  
➗ Division  

⚙ Implemented using **functions and switch-case statements**.

🎯 Purpose: Practice **JavaScript logic and functions**.

---

## 9️⃣ 🐘 PHP Welcome Page with Date & Time
This PHP program displays:

👋 A **Welcome Message**  
📅 Current **Date & Time**

🎯 Purpose: Learn **basic PHP scripting and server-side execution**.

---

## 🔟 📥 PHP Form Handling using POST Method
This program accepts user input from an **HTML form** and displays it using PHP.

🔐 Uses **POST method** for secure data transmission.

🎯 Purpose: Understand **form processing in PHP**.

---

## 1️⃣1️⃣ 🔤 PHP String Manipulation
This program demonstrates **string operations in PHP**.

🔹 Find **string length**  
🔹 **Reverse** a string  
🔹 Extract **substring**

🎯 Purpose: Learn **PHP string functions**.

---

## 1️⃣2️⃣ 📦 PHP Arrays
This program demonstrates **arrays in PHP**.

📚 Features  
- Store multiple values  
- Display values using **loops**  
- Manage data efficiently  

🎯 Purpose: Understand **array data structures in PHP**.
